## About

This lesson contains demonstrations and exercises for building AI agents with persistent long-term memory and evaluating their performance using industry-standard metrics.

- **Demo notebooks**: Fully implemented examples with complete code
- **Exercise solution notebooks**: Complete implementations for reference
- **Exercise starter notebooks**: Partial implementations with TODO comments for you to complete

## Concepts

### Concept 1: Long-Term Agent Memory
Learn how to build AI agents with persistent memory using PostgreSQL, implement intelligent memory management with TTL-based expiration, dynamic weighting, and memory pruning policies.

### Concept 2: Evaluating AI Agents
Master the top 3 agentic RAG evaluation metrics (Factual Accuracy, Citation Compliance, Retrieval Relevance) and learn to integrate LlamaIndex for document retrieval and evaluation.
